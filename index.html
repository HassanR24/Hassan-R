Projects

Twitter Bot Using Python

The goal of the Bot was to check the current internet speed and then compare it to the promised/contract internet speed by the provider. If the speed was below the promised/contract speed, the software is supposed to save the current download and upload speed in a variable, login to Twitter account, start writing a new tweet mentioning the internet provider and speed we are getting comparing it to the speed we were supposed to get and tweeting it. I created the whole script with Selenium to automate the whole process. From opening speed test site, grabbing the required data, opening and logging into the twitter account, writing the message mentioning the internet provider and tweeting it. Things it required the knowledge of was Python, Selenium and HTML

Instagram Bot Using Python.

The purpose of the Bot was to login to Instagram, type a user-provided keyword in search bar, open the first account in search results, open the followers of that account and then finally follow each of the accounts in following lists. I created the whole automation script using Selenium. The bot was created with user's personal chrome profile, undetected chrome driver and random time intervals between each step to avoid bot detection by Instagram.

A Blog Using Flask

I have created a blog using Flask. You can Register to my blog using your email address and password, once logged in, you will be able to post comments on Admin's blog posts. All the new user's passwords are hashed (encrypted). There are certain things only admin accounts are allowed to use i-e creating and deleting a post etc. User's data was initally stored in Flask-SQLAlchemy during the test phase and then converted to Postgres DB after deployment. The blog is currently deployed on "render.com". Flask framework is used including but not limited to Flask Forms, Flask Login, Flask CKEditor Flask-SQLAlchemy and Flask Bootstrap.
